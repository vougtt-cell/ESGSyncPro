# Полная настройка проекта ESG Landing Page (с нуля)
Пошаговая инструкция для запуска проекта локально: установка окружения, база данных, пользователь, запуск сервера и админки.
---
## Шаг 0. Что уже должно быть
- **Windows 10/11** (у тебя уже есть).
- **Node.js** — если нет: скачай LTS с [nodejs.org](https://nodejs.org), установи, в терминале проверь: `node -v` и `npm -v`.
Дальше нужно решить, где будет крутиться база данных: **локально (PostgreSQL на твоём ПК)** или **в облаке (Neon)**.
---
## Вариант A: PostgreSQL на своём компьютере
### 1. Установка PostgreSQL
1. Скачай установщик для Windows: https://www.postgresql.org/download/windows/ (например, с https://www.enterprisedb.com/downloads/postgres-postgresql-downloads).
2. Запусти установщик.
3. Задай пароль для суперпользователя **postgres** — запомни его (например `postgres` для простоты, или свой).
4. Порт оставь **5432**.
5. Остальное можно по умолчанию. Дойди до конца установки.
### 2. Создание базы данных
Открой **Командную строку** или **PowerShell** и выполни:
